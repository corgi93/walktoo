# walkToo 비즈니스 플랜

## 1. 서비스 개요

- **앱 이름**: walkToo
- **카테고리**: 커플 산책 다이어리
- **핵심 가치**: 매일 같이 걸으며, 서로의 하루를 기록하고 열어보는 경험
- **기술 스택**: React Native (Expo) + Supabase

---

## 2. 수익 모델 (BM)

### 프리미엄 구독제

| | 무료 | 프리미엄 |
|---|---|---|
| 산책 기록 | 하루 1개 | 동일 |
| 오늘의 질문 | 매일 1개 랜덤 | 카테고리 직접 선택 |
| 사진 첨부 | 3장 | 5장 |
| 과거 기록 | 최근 30일 | 무제한 |
| 다이어리 테마 | 기본 1종 | 4~5종 (파스텔, 빈티지, 시즌 등) |
| 포토부스 템플릿 | 기본 2종 | 전체 해금 |
| 기념일 리포트 | ❌ | 월간/연간 산책 리포트 |
| 광고 | 없음 | 없음 |

### 가격

| 플랜 | 가격 | 비고 |
|------|------|------|
| 월간 | 990원/월 | 기본 |
| 연간 | 9,900원/년 | 월 825원, 2개월 할인 |

### 핵심 원칙

- **1명 결제 → 커플 둘 다 적용** (거부감 제로, 대화 소재)
- 무료로 핵심 기능 충분히 사용 가능
- 유료는 "더 예쁘게, 더 많이" 영역
- 광고 절대 안 넣음 (감성 앱 톤 유지)

## 3. 인앱결제 기술 검토

### RevenueCat vs 직접 구현

| | RevenueCat | 직접 구현 (expo-iap) |
|---|---|---|
| 구현 난이도 | 쉬움 (SDK 연동) | 어려움 (영수증 검증 직접) |
| 구독 관리 | 자동 (갱신, 해지, 환불) | 직접 구현 |
| 크로스플랫폼 | iOS + Android 통합 | 각각 처리 |
| 대시보드/분석 | 실시간 MRR, 이탈률 등 | 직접 구축 |
| 비용 | 월 $0 ~ $8,500 (매출 기반) | 무료 |
| Expo 호환 | ✅ `react-native-purchases` | ✅ `expo-iap` |

### RevenueCat 요금제

| 플랜 | 조건 | 비용 |
|------|------|------|
| Free | MTR $2,500 이하 | $0 |
| Starter | MTR $2,500 초과 | +1% 수수료 |
| Pro | 추가 기능 | +1% + $8,500/yr |

> MTR = Monthly Tracked Revenue

### 결론: **RevenueCat 추천**

- 초기 무료 (MTR $2,500 = 약 340만원까지 무료)
- 구독 관리 자동화 (갱신, 해지, 그레이스 기간)
- 커플 공유 로직만 Supabase에서 처리하면 됨
- 1인 개발에서 결제 직접 구현은 시간 낭비

## 5. 비용 구조 (20만 MAU 기준)

### 월간 고정비

| 항목 | 월 비용 | 비고 |
|------|--------|------|
| Supabase Pro | 3.4만원 | 기본 플랜 |
| Supabase Auth (추가 MAU) | 44만원 | 10만 초과분 |
| Supabase Storage | 1만원 | 사진 누적 |
| Supabase Bandwidth | 9만원 | ~1TB/월 |
| Supabase DB 확장 | 7만원 | |
| EAS Build Pro | 3만원 | |
| 도메인/기타 | 1만원 | |
| **합계** | **~68만원/월** | |

### 변동비

| 항목 | 비율 | 비고 |
|------|------|------|
| Apple/Google 수수료 | 15% | 소규모 사업자 프로그램 |
| RevenueCat | 0~1% | MTR $2,500 초과 시 |

### 비용 최적화 포인트

- **Supabase Auth MAU가 가장 비쌈** → 50만 이상이면 자체 인증 고려
- **사진 Storage** → Cloudflare R2 전환 시 egress 무료
- **CDN** → Cloudflare로 이미지 캐싱하면 bandwidth 절감

| 채널 | 전략 |
|------|------|
| 인스타그램 | 커플 공감 카드뉴스, 산책 코스 추천 |

### 톤 & 매너

- 개발 비하인드 → 서브 계정/스레드
- 메인은 **커플 타겟 감성** 콘텐츠
- 픽셀아트 일러스트로 차별화

### 초기 유저 확보

1. 지인 커플 10쌍에게 직접 배포
2. 피드백 반영 후 커플 커뮤니티 게시
3. "우리 산책 기록" 공유 기능 → 바이럴
4. 시즌별 이벤트 (벚꽃 시즌, 크리스마스 등)
